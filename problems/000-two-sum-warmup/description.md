# Two Sum Warmup

This is a solved warmup problem so you can get comfortable with the UI.

What to do:

1. Open the code panel and inspect the preloaded solution.
2. Click **Run Tests** to watch the visible tests pass.
3. Click **Submit** to run the hidden case as well.
4. Explore the **Problem Files** panel to inspect or edit the files that define this problem.

Your program receives `TEST_INPUT` as JSON with:

```json
{
  "nums": [2, 7, 11, 15],
  "target": 9
}
```

Print the zero-based pair of indices as JSON, for example:

```json
[0, 1]
```
